# Proyecto Integrador U2 - Gestión de Inventario 📦

Este repositorio contiene el Avance 1 del Proyecto Integrador correspondiente a la **Semana 5** de la asignatura **Desarrollo de Aplicaciones Web**. 

El objetivo principal de esta aplicación es demostrar la implementación de funcionalidades dinámicas en el lado del cliente mediante la manipulación del DOM y el manejo de eventos.

## 🚀 Descripción del Proyecto

Es una aplicación web de una sola página (SPA) diseñada para gestionar de forma rápida un catálogo o inventario. Es ideal para administrar el registro de productos de tiendas en línea, permitiendo clasificar artículos como prendas de vestir, suplementos o accesorios. Toda la interacción ocurre en tiempo real sin necesidad de recargar el navegador.

## ✨ Funcionalidades Principales

*   **Registro Dinámico:** Permite ingresar el nombre, descripción y categoría de un producto.
*   **Validación de Formularios:** Evita el envío de datos vacíos, mostrando alertas dinámicas en pantalla.
*   **Manipulación del DOM:** Crea y renderiza tarjetas de productos en la interfaz de usuario en tiempo real utilizando `createElement()` y `appendChild()`.
*   **Manejo de Eventos (`addEventListener`):** Intercepta el envío del formulario (`preventDefault`) y gestiona las acciones de los botones.
*   **Gestión de Registros:** Incluye la capacidad de eliminar productos individuales de la lista.
*   **Contador en Tiempo Real:** Mantiene un registro visual del total de productos creados y activos en la sesión.

## 🛠️ Tecnologías y Herramientas Utilizadas

*   **HTML5:** Estructura semántica de la aplicación.
*   **CSS3 & Bootstrap 5:** Diseño responsivo (Mobile First) y estilizado de la interfaz.
*   **JavaScript (Vanilla JS):** Lógica de negocio, validaciones y reactividad del lado del cliente.
*   **GitHub Pages:** Despliegue y alojamiento del proyecto web.

## 📱 Visualización (Responsive Design)

El proyecto está construido bajo un enfoque *Mobile First*, garantizando que el formulario y la lista de inventario se adapten perfectamente a resoluciones de dispositivos móviles, tablets y pantallas de escritorio.
